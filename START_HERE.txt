❓ PREGUNTAS FRECUENTES
═════════════════════════════════════════════════════════════════════════════

P: ¿Dónde están los datos de los formularios?
R: Se guardan en la consola del navegador (F12 → Console)
   Para persistencia real necesitarías un backend.

P: ¿Puedo cambiar los colores?
R: Sí, en CSS variables al inicio (--primary, --secondary, etc.)

R: Edita el HTML en talento.html
   Copia/pega secciones y adapta el contenido

P: ¿Funciona sin internet?
R: Sí, la página en tu computadora funciona offline
   Cuando subas a GitHub Pages necesitarás internet

P: ¿Puedo usarlo en producción?
R: Sí, pero para guardar datos necesitarás un backend
   Ahora es solo frontend
