# 🚀 Elijo Quedarme - Talento Joven en Aragón

Una plataforma innovadora diseñada para atraer, retener y conectar talento joven con empresas aragonesas que ofrecen un verdadero valor.

## 📋 Descripción del Proyecto

**Elijo Quedarme** es la solución al desafío de fuga de talento en Aragón. Funciona como una plataforma de certificación y transparencia que permite:

- **Para Jóvenes Talentos**: Descubrir empresas aragonesas reales con estándares verificados de calidad laboral
- **Para Empresas**: Certificarse con sellos de calidad y acceder a una comunidad de talento interesado en quedarse en Aragón

## 🎯 Características Principales

### 💡 Concepto en 3 Pilares

1. **Transparencia Real** 🔍
   - Información verificada sobre ambiente laboral, flexibilidad, sueldos y crecimiento
   - Métricas objetivas comparables

2. **Certificación Verificada** 🏆
   - Sellos de calidad auditados por mentores
   - Auditorías periódicas para mantener los estándares
   - Mayor credibilidad que simples opiniones

3. **Visibilidad para Talento** ⭐
   - Ranking de mejores empresas para trabajar
   - Comparador interactivo
   - Historias reales de empleados que eligieron quedarse

### 📊 Sellos de Calidad

La plataforma certifica empresas en 6 categorías:

| Sello | Descripción |
|-------|------------|
| ❤️ **Bienestar Laboral** | Salud física y mental, espacios cómodos, beneficios de bienestar |
| 🍽️ **Dietas** | Comedor de empresa, ayudas de comida, descansos garantizados |
| 🛡️ **Estabilidad** | Contratos duraderos, baja rotación, carrera a largo plazo |
| 🚀 **Joven e Innovador** | Visión de futuro, tecnología, preparado para cambios |
| 📈 **Desarrollo Profesional** | Formación continua, mentoría, planes de carrera claros |
| 🔐 **Seguridad Laboral** | Normativa, prevención de riesgos, derechos garantizados |

## 🚀 Cómo Usar

1. **Abre el archivo en tu navegador:**
   - Simplemente abre `talento.html` en tu navegador web
   - O sube a GitHub y accede vía GitHub Pages

2. **Características interactivas:**
   - Haz clic en "Soy Joven Talentoso" para registrarte como demandante de empleo
   - Haz clic en "Soy Una Empresa" para registrarte como empresa
   - Los datos del formulario se registran en la consola del navegador
   - Navega suavemente a través de las secciones con los enlaces del menú
