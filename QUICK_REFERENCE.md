## ❓ Preguntas Frecuentes

**¿Puedo modificar los colores?**  
Sí, en CSS variables en `:root`

**¿Cómo agrego imágenes?**  
Pon en carpeta `img/` y usa `src="img/nombre.jpg"`

**¿Funciona sin internet?**  
Sí, es totalmente offline (excepto GitHub Pages)

**¿Puedo usar en producción?**  
Sí, pero necesitarás backend para guardar datos

**¿Cómo agrego más sellos?**  
Copia una `.seal-card` en HTML + CSS si necesitas
