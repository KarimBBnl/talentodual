# Elijo Quedarme - Información del Proyecto

Bienvenido a **Elijo Quedarme**, una plataforma innovadora diseñada para atraer, retener y conectar talento joven con empresas aragonesas.

## 🎯 Misión

Transformar la narrativa del talento joven en Aragón, demostrando que es posible tener carrera profesional, calidad de vida y oportunidades sin necesidad de abandonar el territorio.

## 📊 Datos Clave

### El Reto
- **Problema**: Fuga de talento joven en Aragón
- **Oportunidad**: Diferenciar empresas aragonesas
- **Solución**: Certificación transparente y verificada

### Diferenciadores
✅ Certificación verificada (vs solo opiniones)  
✅ Métricas objetivas (vs impresiones subjetivas)  
✅ Sellos visibles y comparables (vs texto puro)  
✅ Auditoría periódica (vs información estática)  

### Target Audience
1. **Talento Joven** (18-35 años)
   - Recién graduados
   - Profesionales junior
   - Personas en búsqueda de cambio

2. **Empresas Aragonesas**
   - Todas las industrias
   - Todos los tamaños
   - Interesadas en retener talento

## 🚀 Cómo Empezar

### Para Usuarios
1. Abre `talento.html` en navegador
2. Explora las secciones
3. Prueba los formularios de registro
4. Navega con el menú

### Para Desarrolladores
```bash
# Clonar
git clone https://github.com/tu-usuario/Talento_v2.git

# Abrir en navegador (no requiere servidor)
open talento.html

# O iniciar servidor local
python -m http.server 8000

# Luego abrir http://localhost:8000
```
